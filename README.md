# AI Chatbot using Hugging Face Transformers

A conversational AI chatbot built with Hugging Face Transformers that can answer questions, have conversations, and provide information on various topics.

## 🚀 Features

- **Pre-trained Transformer Model**: Uses Qwen/Qwen2.5-7B-Instruct (7 billion parameters)
- **Natural Conversations**: Maintains conversation history for context-aware responses
- **Hugging Face API**: Accesses large language models via Hugging Face Inference API
- **No Hardcoded Responses**: All responses dynamically generated by the AI model
- **Interactive & Automated Modes**: Supports both live chat and scripted demo

## 🛠️ Technologies Used

- **Python**
- **Hugging Face Transformers**
- **Hugging Face Inference API**
- **Jupyter Notebook**
